This site is made with a variety of tools. The source for this site is available in its repository.

  • The frontend is made with 11ty as its static site generator and Nunjucks as its templating engine.
    • The design uses vanilla CSS with some minimal post-processing.
      • The fonts are system fonts and the font stacks are inspired by Modern Font Stacks.
      • The site uses postcss at buildtime to optimize and post-process the CSS.
        • cssnano to minify the CSS for better performance.
        • postcss-custom-media to simplify the responsive CSS declarations.
        • postcss-media-minmax as a polyfill to support CSS Media Queries Level 4
        • postcss-nesting lets you nest style rules inside each other, following the CSS Nesting specification.
        • postcss-preset-env polyfills modern CSS into most something most browsers can interpet.
    • The data is populated through global data files from different services, which are re-fetched at buildtime.
      • The games list is obtained from How Long To Beat.
      • The movies and show list is obtained from Trakt.tv.
      • The manga list is obtained from MangaDex.
      • The music list is obtained from Spotify.
      • The video list is obtained from YouTube.
  • The site is inspired by some fellow authors:
    • PhotoGabble, for features such as this colophon or some 11ty plugins.
    • Cory Dransfeldt, for inspiring the media and recent activity sections.
