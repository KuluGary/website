A colophon page explaining the tools, technologies and principles governing this website.

The source for this site is available in the Github repository.

Philosophy

When I started this website circa 2003, I tried to build it in a way that aligned with my priorities:

  • Provide an accessible and inclusive reading experience for most people. This includes using semantic HTML and try to be a11y compliant, minimize the weight of the site to decrease the data use, and create a welcoming atmosphere.
  • Make a personal website that reflects myself, my interests and my preferred aesthetics.
  • Learn and implement more IndieWeb protocols and create a site as independant from corporate or business owned platforms as possible.

Technologies

This website is staticly generated with 11ty and Nunjucks, and it uses both HTML template files and Markdown files.

It uses basic CSS as much as is comfortable, but there's some downleveling of new CSS standards with PostCSS.

The site is hosted on Neocities as static HTML files and its repository is in Github. The site is deployed through a Github action.

Webmentions are fetched from webmention.io and connected to social media with brid.gy. They are sent thanks to a bash script inspired by localghost.dev at buildtime.

Design

The base template for the website layout is from The Monospace Web. At this point in time, it's been heavily modifided.

The site uses system fonts to minimize page size and loading times, and every option available at the settings page is from Modern Font Stacks.

See more about the design in the styleguide page.

The data

The data is populated through global data files from different services, which are re-fetched at buildtime.

  • The game list is obtained from How Long To Beat.
  • The movie and show list is obtained from Trakt.tv.
  • The manga list is obtained from MangaDex.
  • The music list is obtained from Spotify.
  • The video list is obtained from YouTube.
  • The webcomics are obtained through a series of third-party sites and manually compiled into a JSON file.

Inspiration

The site is inspired by some fellow web-authors:

  • PhotoGabble, for features such as this colophon or some 11ty plugins.
  • Cory Dransfeldt, for inspiring the media and recent activity sections.
  • Sophie Koonin, for inspiring improvements to my webmentions.
