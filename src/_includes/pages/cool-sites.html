Cool sites

I've been around for a while, and I've seen some websites. These are just a few of my favourites.

32-Bit cafe If you miss the old-school feel of the Internet before it was profitable for corporations, check out the info and resources provided by the 32-Bit cafe.
Vimm's Lair Some of my favourite games of all time are now considered retro, and if you are into game preservation you'll find that Vimm's Lair has an enormous database you can browse through.
Azgaar's Fantasy Map Generator Fantasy maps is something I love looking at, from Lord of the Rings to The Witcher. But if you ever wanted to make one for yourself, check out Azgaar's.
Watabou's Procgen Arcana If you liked Azhaar's, Watabou's offers an array of different map generators from cities, villages and dungeons.
Kenney's Game Assets Kenney has ammassed a gigantic collection of free-to-use game assets, from 2D to 3D to UI.
the museum of alexandra Alexandra's personal website is an inspiration in both style and the sheer amount of content shared. If you want inspiration to start your own personal site, be sure to check hers out.
The Dungeon Master's Vault An online service for D&D5e players and Dungeon Masters to generate character sheet and other resources.
Mythological Figures (for D&D5e) A collection of NPC stat blocks based on mythological and historic figures.
NPC Statblock Compendium A PDF full of NPC stat blocks based on character classes and races for D&D5e.
Godot Shaders A community driven shader library for the Godot game engine.
Game design wiki A collection of resources for game designers.
How Long To Beat If you want to manage your game backlog or share what you are playing, this site is awesome. I'm currently using it to manage my /now page.
Game accessibility guidelines If you are a game developer and are concerned with the accessibility of your games, you should check out this site.
Game UI Database A comprehensive database and tool for game interface designers. This site is awesome if you are in need for inspiration for your games' UI.
