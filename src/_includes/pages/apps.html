Software

I'm currently using a Windows 10 machine and the software I use is installed on that operating system, so availability may vary.

  • Firefox: to browse the web.
  • Godot: to make videogames.
  • Visual Studio Code: to write code.
  • GIMP: to make and edit photos and images.
  • Inkscape: to make vector graphics.
  • Krita: to make digital art.
  • Blender: to make 3D models.
  • Libresprite: to make pixel art.
  • LibreOffice: for creative writing.
  • OBS: to make screen recordings and livestreams.
  • Foxit PDF Reader: to read PDFs.
